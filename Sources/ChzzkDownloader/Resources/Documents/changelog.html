변경 사항

1.2.0

  • 대시보드에서 현재 방송 중인 채널의 카테고리와 태그를 확인할 수 있습니다.
  • 채널마다 "방송 중 태그가 바뀌어 일치하지 않으면 녹화 중단" 옵션을 추가했습니다. 켜두면 녹화 중 태그가 필터와 다르게 바뀔 경우 파일을 안전하게 저장하고 녹화를 중단합니다.
  • 앱이 비정상 종료될 때 남아 있던 .part 파일을 다음 실행 시 자동으로 복구합니다.
  • 녹화 종료 후 ffmpeg가 5초 내로 종료되지 않으면 강제로 종료하여 좀비 프로세스를 방지합니다.
  • 성인 방송 또는 재생 정보가 없는 방송에서 과도하게 재시도하던 문제를 수정했습니다.
  • 인증 실패 감지 로직을 개선하여 일반 로그 메시지나 CDN URL을 오탐하지 않습니다.

1.1.0

  • 감시 중이던 채널의 상태를 저장했다가 앱을 다시 켰을 때 자동으로 감시를 재개합니다.
  • 채널마다 녹화 태그를 설정하여 방송 태그가 일치할 때만 녹화하도록 할 수 있습니다.

1.0.3

  • 도움말 > 릴리즈 노트 화면에서 변경 사항을 업데이트 알림과 동일한 형식으로 볼 수 있습니다.
  • 자동 녹화 안정성을 추가로 개선했습니다.

1.0.2

  • 자동 녹화 안정성을 크게 개선했습니다.
    • 방송 중 일시적인 네트워크·스트림 오류로 녹화가 멈추던 문제를 자동으로 복구합니다.
    • 스트림 연결에 실패하면 곧바로 포기하지 않고 여러 번 다시 시도합니다.
    • 장시간 녹화 중 스트림 토큰 갱신이 정상적으로 동작합니다.
    • 녹화가 시작 직후 끊기면 즉시 다시 시도해 긴 공백을 없앴습니다.
  • 저장 폴더를 쓸 수 없을 때(예: 외장 드라이브 분리) 명확한 안내를 표시합니다.

1.0.1

  • 최신 streamlink 버전에서 라이브 녹화가 도중에 멈추던 문제를 수정했습니다.

1.0.0

  • 첫 출시 — 라이브 자동 녹화, VOD·클립 다운로드, 예약 녹화, 파일 분할, 다운로드 기록, 브라우저 쿠키 불러오기.
